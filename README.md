# TimeCollieMini
C++ 콘솔 기반 일정 관리 프로그램입니다.

개인 일정을 등록, 조회, 수정, 삭제할 수 있으며
월간 달력 보기와 반복 일정 관리 기능을 포함한 미니 스케줄러입니다.

개발 기간

2026.06.01 ~ 2026.06.03

사용 기술
C++17
STL Vector
File I/O
주요 기능
일정 추가
일정 수정
일정 삭제
날짜별 조회
월간 달력 보기
반복 일정 관리
파일 저장 / 불러오기
실행 화면
메인 화면




월간 달력 보기




전체 일정 조회




반복 일정 추가




실행 방법

TimeCollieMini.exe 파일을 실행하면 콘솔에서 프로그램을 사용할 수 있습니다.

제한 사항
콘솔 기반 프로그램이므로 알림 팝업 기능은 없습니다.
반복 일정의 특정 회차만 수정하거나 삭제하는 기능은 구현하지 않았습니다.
